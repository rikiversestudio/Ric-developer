Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat dan menyusun halaman website. HTML menggunakan elemen atau tag untuk memberitahu browser bagaimana menampilkan konten.

Struktur Dasar HTML

Berikut struktur umum file HTML:

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Halo Dunia</h1>
    <p>Ini adalah paragraf pertama saya.</p>
  </body>
</html>

Tag Umum yang Wajib Diketahui

  • <h1> - <h6>: Judul (semakin kecil angkanya, semakin kecil ukurannya)
  • <p>: Paragraf
  • <a>: Link (tautan)
  • <img>: Gambar
  • <ul>, <ol>, <li>: Daftar
  • <div>: Kotak umum
  • <span>: Teks kecil (inline)

Contoh Membuat Link

<a href="https://www.youtube.com/@RikiVerseStudio" target="_blank">Kunjungi Channel YouTube-ku</a>

Contoh Menampilkan Gambar

<img src="https://via.placeholder.com/300x200" alt="Contoh Gambar">

Penutup
